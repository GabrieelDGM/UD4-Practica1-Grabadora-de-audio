# Ejercicio 6 - Animación propia.

## Objetivo.
Se creo una animación propia para los botones de Iniciar y Stop, creando un componentente `AnimationBotton`, añadiendo el efecto en ambos botones.

## Funcionamiento.
Cuando iniciamos la aplicacion los botones se ven de tamaño normal pero cuando lo pulsamos cambia el tamaño del boton y comienza la animacion de manera fluida y atractiva visualmente.

## Librerías utilizadas
* Animated de React Native → para la animación de escala
* useRef de React → para mantener el valor de la animación entre renders
